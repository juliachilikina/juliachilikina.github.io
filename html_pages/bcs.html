Bootcamp Spot

redesign

Bootcampspot is an educational space for UC Berkeley Extension students and faculty. Students access this site to check-in to class, submit homework, check grades, view upcoming events and check their peogress. Being bootcampers ourselves we'd been using BCS for 6 months and knew its weaknesses. So for our final project, we decided to redesign bootcampspot.

We wanted to create a platform that is focused on the user experience. With all of the useful info on BCS, we want it to be the tool that students use for more than just checking in and submitting homework.

Type

UX/UI, web design

Time

2 weeks

My role

UX/UI designer

Overview

The current layout of the site is not optimized for the tasks that are performed most frequently. We want to address this problem by redesigning the structure and layout of bootcampspot. Creating better information hierarchy, user flow, and visual design might improve the student experience.



Goals

  • Find user pain points

  • Understand which features students access most

  • Discover any features students want in a hub

Paper prototype & Hi-Fi

Ideating led us to different visions for the layout, and what can better calibrate what’s best for our users than A/B testing? We put both of these products in front of different testers, and took away the best of both, for instance ability to join remotely from the homepage and personal stats that stand out (version on the left), customized calendar on the dashboard and progress bar (version on the right). With this findings we are ready to move to lo-fi prototyping and wireframing.

Lo-Fi prototypes and wireframes

Based on A/B testing from paper prototype, we took what worked well and implemented into lofi which we spent the most time in this process cuz we would like to make sure that we deliver the smoothest use flow before we move on to the next design process. Design system

Our design system consists of the fonts, colors, interface elements, and sample imagery that communicate the essence of our visual brand. We wanted to create a platform that invites user to stay and explore. Our dark blue primary color is balanced with bright accents and a lot of negative space. We use a super family font Poppins for 2 main reasons: It’s a geometric sans-serif that has big round characters and good spacing. Its big lower letters relative to the uppercase keeps the font from being boring and what is more important gives high readability. And, we wanted to use a google font because it works on all platforms in chrome Here’s our UI elements and components, that include primary and secondary buttons Click and a set of icons that we created To be sure that we can move on with the chosen options we conducted a color blindness test. Here you can see how people with 4 different types of visual impairment would see our stile tile.
